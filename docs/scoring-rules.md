# AthenaXI — Scoring Rules

## Batting
| Action | Points |
|--------|--------|
| Run scored | 1 |
| Boundary (4) | +1 bonus |
| Six (6) | +2 bonus |
| 50 runs | +10 bonus |
| 100 runs | +20 bonus |
| Duck (out for 0) | -5 |

## Bowling
| Action | Points |
|--------|--------|

| Maiden over | +10 |
| 3-wicket haul | +15 bonus |
| 5-wicket haul | +30 bonus |

|------|-----------|
